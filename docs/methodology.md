# Методология

# 1. Общая схема эксперимента

Для каждого режима температуры выполняются следующие шаги:

1. Обучение модели с фиксированным seed;
2. Сохранение финального чекпоинта;
3. Построение пути между двумя состояниями модели;
4. Оценка высоты барьера $\Delta L$;
5. Оценка прокси-кривизны $\kappa_{\text{tr}}$;
6. Сравнение обобщающей способности

Все эксперименты повторяются для нескольких случайных seed

# 2. Модель и датасет

Используется классификация CIFAR-10
с архитектурой ResNet-18

Функция потерь — cross-entropy

Валидационный split фиксирован и одинаков для всех запусков

# 3. Интервенции температуры

Температура обучения трактуется как уровень стохастичности
динамики оптимизации

В работе варьируются:

- скорость обучения,
- размер батча,
- (при необходимости) learning rate

Каждый режим температуры определяется набором гиперпараметров
$\mathcal{T}_k$

# 4. Обучение

Для каждого режима $\mathcal{T}_k$
и для каждого seed $s_i$
обучается модель, после чего получаем состояния:

$$
\theta^{(k, s_i)}
$$

Случайность фиксируется через:

- seed генераторов,
- фиксированный split данных

# 5. Протокол интерполяции

Для двух состояний
$\theta^{(k, s_i)}$ и $\theta^{(k, s_j)}$
строится линейный путь:

$$
\theta(t) = (1-t)\theta^{(k, s_i)} + t\theta^{(k, s_j)}
$$

Путь дискретизируется равномерной сеткой по $t$

В каждой точке оценивается:

$$
L(t) = L(\theta(t))
$$

Перед оценкой при необходимости пересчитываются
BatchNorm-статистики

# 6. Оценка барьера

Высота барьера $\Delta L$
вычисляется согласно определениям,
приведённым в разделе Definitions

Для набора порогов $\epsilon$
вычисляется индикатор:

$$
\text{jump}_\epsilon =
\mathbf{1}\{\Delta L > \epsilon\}
$$

# 7. Геометрия минимума

Для каждого состояния $\theta^{(k,s_i)}$
оценивается прокси локальной кривизны:

$$
\kappa_{\text{tr}}
$$

Оценка производится по случайным направлениям
методом конечных разностей

# 8. Метрики обобщения

Для каждого состояния фиксируются:

- значение $L(\theta)$,
- точность классификации

# 9. Агрегация результатов

Для каждого режима температуры $\mathcal{T}_k$
вычисляются:

- средняя высота барьера,
- доля случаев $\text{jump}_\epsilon = 1$,
- среднее значение $\kappa_{\text{tr}}$,
- средняя точность

Это позволяет сравнить:

- вероятность перехода между долинами,
- геометрию минимумов,
- качество обобщения
